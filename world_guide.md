# COrebound LAttence 世界觀與視覺圖鑑 (World Setting & Visual Guide)

本文件記載了 **COrebound LAttence (COLA)** 的世界設定。在本作中，玩家所探索的四個主題世界並非傳統地理疆域，而是代表著「圖騰的核心繪畫語言（Drawing Style）」。視覺圖騰的編織畫法直接塑造了該世界的生態、環境與文明型態。

---

## 1. 幾何世界 (Geometry)

幾何世界象徵著高度理性的規律性與人造文明的演進。在此世界中，一切視覺圖騰不是簡單的形狀變化，而是透過四個階級展示出不同層次的**「數學之美 (Mathematical Aesthetics)」**：

* **核心象徵**：科技文明、Corebound 核心力學、Lattice 晶格網格。
* **數學美學與圖騰階級 (Tier Hierarchy)**：
  * **Tier 1: Euclidean (歐氏幾何) —— 最純粹的基礎幾何**
    * *設計理念*：玩家一眼即可理解的幾何起源，呈現極致的秩序與簡潔。
    * *代表圖騰*：
      1. **圓形 (Circle)**：代表完美、旋轉與循環。
      2. **三角形 (Triangle)**：代表穩定、尖銳與張力。
      3. **正方形 (Square)**：代表秩序、理智與人造文明。
      4. **正六角形 (Hexagon)**：代表自然界的高效蜂巢與晶格結構。
  * **Tier 2: Symmetry (對稱幾何) —— 開始顯現的經典神聖圖樣**
    * *設計理念*：利用數學重複與對稱，營造讓玩家驚嘆「哇」的經典神聖圖形。
    * *代表圖騰*：
      1. **彭羅斯三角 (Penrose Triangle)**：不可能三角，挑戰空間維度的幾何矛盾。
      2. **梅塔特隆立方體 (Metatron Cube)**：包含所有柏拉圖立體的幾何神聖圖樣。
      3. **魚形橢圓 (Vesica Piscis)**：雙圓交集，眾多神聖幾何與比例的開端。
      4. **生命之花 (Flower of Life)**：多重交疊的同心圓網格，完美的抽象圖騰編織。
  * **Tier 3: Emergence (湧現幾何) —— 湧現與生命複雜度的交會**
    * *設計理念*：由許多簡單的幾何數學規則，繁衍組合出的複雜自組織結構。
    * *代表圖騰*：
      1. **室利仰陀羅 (Sri Yantra)**：印度經典神聖幾何，九個交錯三角形交織的複雜圖騰。
      2. **曼德博集合 (Mandelbrot - 簡化版)**：展現無限自相似性的數學分形圖樣。
      3. **阿波羅尼奧斯墊片 (Apollonian Gasket)**：在限定空間內無限塞圓的完美極限之美。
      4. **沃羅諾伊胞腔 (Voronoi Cell)**：基於點距劃分的胞腔多邊形，代表細胞與自然裂紋。
  * **Tier 4: Higher Dimension (高維幾何) —— 超越三維理解的終極融合**
    * *設計理念*：世界 Boss 專屬。這裏的圖騰不再是變大，而是維度的「融合與完整」，展現超越三維視角的超維幾何生命。
    * *代表圖騰*：
      1. **600胞體 (600-cell)**：四維空間的正多胞體，數學與幾何的究極形體。
      2. **超立方體 (Tesseract)**：投影於平面上的超次元立方體結構。
      3. **超立方曼陀羅 (Hypercube Mandala)**：融合超維度投影與東方曼陀羅對稱性的自創圖徽。
      4. **核心晶格 (Core Lattice)**：整個 Corebound Lattice 本身即是終極的幾何生命體。
* **玩家視覺感受**：強烈的人為雕琢感與數學秩序，從基礎圖形逐步演進到超維度幾何的視覺震撼。

## 2. 有機世界 (Organic)

有機世界是生命的溫床，象徵著自然界最原始、最茂盛的生長力。在此世界中，生長不以傳統的實體動植物（如綠樹或猛獸）形式出現，而是以**「生命尺度 (Life Scale)」**作為演進主軸。隨著階級提升，圖騰在視覺上呈現從最微觀的生命印記，逐步演化為宏觀的自然現象：

* **核心象徵**：生命的脈動、生態系統的交織、非具象的生長現象。
* **生命尺度與圖騰階級 (Tier Hierarchy)**：
  * **Tier 1: 生命單元 (Life Unit) —— 禪繞畫筆法：【線 (Line)】**
    * *設計理念*：玩家看到的是生命最微觀、最細小的基礎印記。
    * *代表圖騰*：
      1. **葉脈 (Leaf Veins)**：細密分叉的營養管道線條，最純粹的禪繞筆觸。
      2. **羽毛 (Feather)**：輕盈且高重複性的生物紋路。
      3. **苔蘚 (Moss)**：微小密集的點狀圖案。
      4. **菌絲 (Mycelium Threads)**：開始向外蔓延的細微網路絲線。
      5. **種子 (Seed)**：細小但蘊含龐大生命力爆發點的圓形徽記。
  * **Tier 2: 生命組織 (Life Structure) —— 禪繞畫筆法：【分支 (Branch)】**
    * *設計理念*：單元開始聚集並分叉，形成具備初步功能與放射對稱的生命結構。
    * *代表圖騰*：
      1. **樹枝 (Branches)**：生命朝空間探索的分叉軌跡。
      2. **珊瑚 (Coral)**：鈣化堆疊的有機幾何骨架，極具圖騰裝飾美感。
      3. **蕨類 (Fern)**：具有完美幾何自相似性的分形葉片。
      4. **蟲巢 (Insect Nest)**：由無數微小孔洞構成的生物群落結構。
      5. **花瓣 (Flower Petals)**：展現完美中心放射性的有機結構。
  * **Tier 3: 生命系統 (Life System) —— 禪繞畫筆法：【網路 (Network)】**
    * *設計理念*：生命組織彼此編織，不再是單一生命體，而是一整片自成體系的森林與生態網。
    * *代表圖騰*：
      1. **樹冠 (Canopy)**：密集成蔭、交織重疊的頂端生態網。
      2. **河流 (Rivers)**：宛如大地血管般奔流分流的水流紋格。
      3. **菌落 (Colony)**：在平面上無休止拓展開來的複雜紋格系統。
      4. **根系 (Root System)**：深埋地底、縱橫交錯的地下脈絡。
      5. **蜂巢 (Hive)**：非歐氏幾何，而是以生物本能築成的六角形生命結構。
  * **Tier 4: 生命現象 (Living Phenomenon) —— 禪繞畫筆法：【流動 (Flow)】**
    * *設計理念*：世界 Boss 專屬。Boss 不再是一隻巨大的怪獸或大樹，而是「生命正在發生」的宏觀自然現象與動態流體。
    * *代表圖騰*：
      1. **颱風 (Typhoon)**：並非氣象風暴，而是由數萬條藤蔓、葉片與羽毛共同交織旋轉的巨大生命漩渦。
      2. **海嘯 (Tsunami)**：無數交錯的葉脈與水流紋路，像大地的血液集體奔流與傾瀉。
      3. **菌爆 (Mycelium Burst)**：地底萬千菌絲在瞬間大範圍破土爆發，形成覆蓋視野的編織網絡。
      4. **花開 (Blossom)**：視野所及之處，數以萬計的圖騰之花在同一瞬間集體綻放，展現大範圍視覺色彩衝擊。
      5. **落葉漩渦 (Leaf Vortex)**：數千片羽毛與樹葉圖騰以極高角速度在空中打轉、相撞並重新解構的動態編織。
* **玩家視覺感受**：充滿生命力、不斷呼吸與搏動的「活體生態圈」。從小巧的筆觸線條，層層交織成極具視覺震撼力的大自然動態流動。

## 3. 分形世界 (Fractal)

分形世界是無序與有序的交界處，象徵著魔法與混沌的深淵。在此世界中，生長與複雜度遵循着**「自相似性 (Self-Similarity)」**的鐵律。升級並非使圖案單純地放大，而是讓圖形無限地自我複製、嵌套與生長：

* **核心象徵**：混沌之力、無限遞歸、深淵魔法、不可終止的循環。
* **分形尺度與圖騰階級 (Tier Hierarchy)**：
  * **Tier 1: 母圖形 (Mother Pattern) —— 初始起點**
    * *設計理念*：玩家看到的是分形的最初代發生器（Generator），只有單一的基礎母圖案。
    * *代表圖騰*：單一的基礎幾何或有機形體（例如一個純粹的五角星 ⭐，或一個最簡單的三角形），乾淨且毫無分叉。
  * **Tier 2: 初級分形 (First-order Fractal) —— 頂點分叉**
    * *設計理念*：母圖案的每一個端點或角落，都開始生長出與母圖案結構相同但尺寸更小的子圖案。
    * *代表圖騰*：在 1 星母五角星的五個頂點上，再次延伸生長出五個微型五角星（如五角星幾何的初次繁衍）。
  * **Tier 3: 次級分形 (Second-order Fractal) —— 系統湧現**
    * *設計理念*：每個子圖案的角再次開始無休止生長，呈現經典的複雜分形圖案。
    * *代表圖騰*：
      1. **科赫雪花 (Koch Snowflake)**：線段不斷以等邊三角形分叉，最終形成雪花邊界的複雜結晶。
      2. **謝爾賓斯基三角形 (Sierpinski Triangle)**：在三角形內部不斷挖出反向三角形的無限鏤空圖樣。
      3. **龍形曲線 (Dragon Curve)**：透過重複對摺而展現出的自我嵌套折線，具備極強的律動美感。
  * **Tier 4: 終極維度 (Infinite Recursion) —— 無盡深淵**
    * *設計理念*：世界 Boss 專屬。這裏沒有起點與終點的概念。Boss 本身就是一個無限大且無限小的集合體，玩家看到的不是獨立怪物，而是整個視野、地圖與 Boss 都是由同一個微小圖案無限嵌套而成的分形矩陣。
    * *代表圖騰*：無限重複、無法觀測出初始核心、完全與世界背景背景融為一體的分形網格，將玩家包裹在無盡的幾何魔法深淵中。
* **玩家視覺感受**：充滿神祕、深邃且令人眩暈的「幾何魔法」，在重複與自我複製中展現超越空間維度的宏大感。

## 4. 機械世界 (Mechanical)

機械世界象徵著工業時代的粗獷與精密。蒸氣與齒輪的咬合驅動著整個世界的運轉，地圖本身就是一座巨大無比的工廠。

* **核心象徵**：重工業、蒸汽龐克、精密機械學。
* **視覺特徵**：
  * **齒輪 (Gears)** 與 **軸承 (Bearings)**：力量傳遞的機械結構。
  * **螺絲 (Screws)**、**管線 (Pipes)** 與 **電路 (Circuits)**：將動能與信號導向各處的基礎通路。
* **玩家視覺感受**：充斥著金屬光澤、黃銅質感、蒸氣噴發與冰冷機械結構的「工業心臟」。

## 5. 敵方單位（圖騰形體）視覺與層級規範 (Enemy Visuals & Tier Specifications)

本作的敵方單位（小怪與 Boss）並非傳統遊戲中具體化、肉身化或「屍體化」的怪獸。牠們在本質上是**模仿玩家小隊結構的抽象禪繞畫花紋形體**。一切戰鬥皆圍繞著中心的小隊圓圈展開，敵方單位同樣是由各個主題世界的獨特繪圖風格（Geometry, Organic, Fractal, Mechanical）所交織出的同心圓花紋。

### 5.1 敵方圖騰層級與圓圈結構 (Ring Structure)
敵方單位的強度與威脅級別（Tier 1 - Tier 4）由其**同心圓圈的數量**決定，層級越高，花紋越複雜，形體越接近玩家的完整小隊：

* **Tier 1 (T1) - 單圈圖騰**：
  * **結構**：僅有**最中央的一圈**圓圈。
  * **視覺**：最基礎的單一畫風花紋，形體小而純粹。
* **Tier 2 (T2) - 雙圈圖騰**：
  * **結構**：由**兩圈**同心圓組成。
  * **視覺**：花紋開始出現內外層的層次對比與交錯，視覺複雜度提升。
* **Tier 3 (T3) - 三圈圖騰**：
  * **結構**：由**三圈**同心圓組成。
  * **視覺**：具備中等複雜的幾何、有機或機械嵌套結構，具備明顯的戰略威脅感。
* **Tier 4 (T4) - 四圈圖騰 (世界 Boss)**：
  * **結構**：由**四圈**同心圓組成。
  * **視覺**：形體與細節最為繁複，視覺大小與複雜度基本與玩家的完整小隊（四圈同心圓禪繞畫）相當。展現該繪圖世界最極致的視覺張力。

*(備註：目前遊戲未設定 Tier 5 層級。)*

## 6. 設計總結

這四個世界在遊戲設計中，不僅僅是背景地圖的抽換，更是小隊與怪物圖騰繪製樣式（Zentangle）的基礎筆法。角色在不同世界的戰鬥，就是將這四種獨特的「畫法」與敵我雙方的「同心圓圖騰」交織在一起的動態過程。
