# 第一次遊玩教學流程草稿

本文件用來記錄第一次遊玩時的教學流程討論。它先不追求完整實作規格，而是把玩家第一次接觸遊戲時，應該先理解的畫面元素、用語、敘事順序與 UI 引導目標整理下來。

目前已定方向：

- 第一次沒有遊玩紀錄時，New Game 會自動開啟教學模式。
- 教學模式預設開場方式為 No Starting Boost。
- 玩家仍可在準備頁改選 Showcase。
- 玩家正式進場一次後，之後 New Game 才顯示「需要教學嗎？」切換。
- Tutorial 不是一般難度，而是比 Easy 更低的完整引導難度。
- Tutorial 的目標是手把手帶玩家完成一次完整通關，而不是測試玩家技術。
- Tutorial 會以一連串支線任務/主線任務的方式推進，讓玩家照著任務逐步跑完整個遊戲循環。

---

## 0. Tutorial 的整體定位：最低難度的任務式通關

Tutorial 應該被視為一種特殊難度層，而不是單純的新手提示。

如果之後遊戲有難度選擇，概念可以是：

| 難度 | 定位 |
| --- | --- |
| Tutorial | 比 Easy 更低。手把手帶玩家完成一次完整通關。 |
| Easy | 給已理解基本操作、但還不想承受太大壓力的玩家。 |
| Normal | 預設完整體驗，要求玩家自行規劃資源與戰鬥路線。 |
| Hard 以上 | 提高敵人壓力、資源壓力、Boss 機制與通關路線要求。 |

Tutorial 的核心不是「把所有規則講完」，而是讓玩家在幾乎不需要高技術的情況下，完成一次從開場到通關的完整流程。

它更像任務導覽：

- 玩家照著任務走，就能知道下一步該做什麼。
- 每一個任務只教一個主要概念。
- 必要時系統可以降低敵人壓力、補足資源、標記目標位置，甚至暫時限制玩家走錯方向。
- 教學通關後，玩家才真正理解普通難度裡「自己規劃路線」是什麼意思。

### 0.1 Tutorial 的任務設計原則

每個階段都應該符合以下節奏：

1. **先給目標**：玩家知道現在要做什麼。
2. **再給操作**：玩家知道要按什麼、去哪裡、碰什麼。
3. **完成後解釋意義**：玩家知道自己剛剛學到哪個系統。
4. **最後預告下一步**：玩家知道下一個任務為什麼自然接上來。

任務不要一次教太多。Tutorial 應該像有人牽著玩家走完第一趟旅程，而不是把一本說明書貼在玩家臉上。這點很關鍵，因為本作的系統密度很高；教學如果太像百科，玩家會被知識量壓扁。

## 1. 教學第一段：映入眼簾的旋轉圓圈

### 1.1 玩家第一眼要被告知什麼

玩家進入遊戲後，第一個需要理解的不是按鍵，也不是數值，而是畫面中央那個正在旋轉的圓形圖騰。

這個圖騰在世界觀裡稱為 **Lattice**。

Lattice 不是單純的裝飾，也不是普通血條。它同時代表：

- 玩家目前的小隊本體。
- 隊長與隊員之間的陣型關係。
- 隊伍升級、招募與成長後留下的視覺痕跡。
- 玩家在這個世界中的存在方式。

教學第一段要讓玩家知道：「我不是在操作一個單人角色，我是在培養、移動、保護一個會成長的 Lattice。」

### 1.2 建議教學文案草稿

> 你眼前旋轉的圓形圖騰，稱為 Lattice。
>
> Lattice 是你的小隊在這個世界中的形狀。它會隨著隊員、陣型與升級逐漸改變。
>
> 中央是隊長，也是整個 Lattice 的核心。只要核心倒下，這一局就會結束。
>
> 圍繞核心的三名初始隊員，是你第一層小隊力量。他們負責保護、輸出與支援，並且會在之後透過升級與招募，讓整個 Lattice 變得更完整。

### 1.3 這三個人代表什麼

第一次教學中，畫面上的三名初始隊員要被解釋成「Lattice 的第一個骨架」，而不是單純三張角色卡。

三名初始隊員可以用以下方式介紹：

- **內層隊員**：靠近隊長，較安全，代表保護核心與穩定起步。
- **中層隊員**：承接內外層，代表隊伍的平衡與轉換空間。
- **外層隊員**：最接近外部威脅，代表第一線接觸、輸出與承傷。

這一段不需要一開始就講完整職責倍率。第一次玩家還在認圖，不要立刻丟數字。先讓他理解「位置有意義」就好。

### 1.4 升級時 Lattice 會怎麼變

玩家需要被告知：每次升級不只是數值變強，角色所屬的圖紋也會變化。

建議用語：

- 每位隊員都有自己的圖紋。
- 隊員升級後，圖紋會變得更明顯、更複雜。
- Lattice 會把這些圖紋編織進整體外觀。
- 因此玩家的隊伍不是只在數字上成長，也會在畫面上留下自己的形狀。

這是 COrebound LAttence 很重要的第一印象：成長不是只看面板，而是「你養出的圖騰真的變了」。

### 1.5 招募其他隊員的意義

玩家也要在第一段知道，Lattice 不是固定三個人。

教學可以這樣接：

> 你現在只看見 Lattice 的起始形狀。
>
> 在探索世界時，你可以招募其他隊員，讓他們加入不同層位。新的隊員會帶來新的能力，也會把新的圖紋加入 Lattice。
>
> 隊伍越完整，Lattice 就越強，也越接近你自己的戰鬥風格。

這一段的目的不是立刻教玩家怎麼招募，而是先埋下動機：「我等一下探索世界，是為了讓這個圖騰變完整。」

## 2. 教學第二段：移動與第一次碰觸敵人

### 2.1 玩家要做的第一個動作

介紹完 Lattice 之後，玩家需要立刻獲得控制感。

這一段的教學目標很單純：讓玩家知道 Lattice 會作為一整個小隊一起移動。

建議提示：

> 使用 WASD 或方向鍵移動 Lattice。
>
> 你不是在移動單一角色，而是在移動整個小隊圖騰。

如果要做成 UI 任務，可以設為：

- 任務文字：`使用 WASD 或方向鍵移動`
- 完成條件：玩家任意方向移動超過短距離。
- 完成回饋：Lattice 外圈輕微亮起，提示玩家「整個圖騰都被你帶動了」。

### 2.2 引導玩家碰觸黑白敵人

移動完成後，下一個教學目標是讓玩家主動接觸第一個黑白敵人。

這裡的提示不要急著把整套世界觀全部講出來。第一次只要讓玩家知道：

- 黑白的是敵人。
- 它們是這個世界自然產生的生命體。
- 它們象徵一切連結中排列組合的構造。
- 玩家可以用 Lattice 的外層去碰觸它們，理解碰撞戰鬥的基本規則。

建議提示：

> 前方黑白的生命體是敵人。
>
> 先靠近它，讓 Lattice 的外層碰到它。

可選的世界觀補充，不建議第一時間全顯示：

> 這些生命體是世界自然生成的排列構造。它們不是普通野獸，而是連結、圖紋與規則在世界裡長出的形狀。

COLA 的部分此時只留在設計文件備註中，不在遊戲內第一段明講：

- 一切排列組合的極大成，最終會指向 COLA。
- 但第一次戰鬥教學不應直接說明 COLA，否則會過早暴露終局概念，也會讓玩家在還沒學會走路時先背世界史。

### 2.3 第一次進入攻擊範圍的規則

玩家碰到黑白敵人時，要先解釋最基礎的攻擊範圍概念：

> 當 Lattice 的某一層與敵人重疊時，就代表進入攻擊範圍。

更白話的提示版本：

> 外層碰到敵人時，外層正在承受風險，也會開始準備攻擊。
>
> 用外層接敵，比讓敵人碰到核心安全。

這段教學先只教「進入攻擊範圍」。不要在這一刻同時教職責倍率、護盾、治療、控制效果。玩家只要先建立一個本能：

`讓外層去碰敵人，不要讓敵人碰核心。`

設計備註：之後我方與敵方的攻擊頻率應該可以拆開，不必設計成雙方同時受傷或同時攻擊。教學文字要避免把「同時互傷」說死，而是先用「進入攻擊範圍」和「我方攻擊頻率」來承接。

### 2.4 建議互動節奏

第二段可以拆成 4 個短任務：

1. **移動 Lattice**：按 WASD 或方向鍵，讓玩家感覺整個圖騰被自己帶動。
2. **看見黑白敵人**：高亮前方敵人，文字只說「黑白的是敵人」。
3. **用外層碰觸敵人**：引導玩家用 Lattice 外層接觸敵人。
4. **理解攻擊範圍**：顯示外層進入攻擊範圍，接著把注意力導向 HUD 的攻擊計時條。

### 2.5 下一段預告：HUD

碰撞完成後，不要立刻進入複雜養成。下一段應該介紹 HUD，因為玩家剛剛已經看見自己受傷，心中自然會問：

- 我現在還剩多少生命？
- 哪一層受傷？
- 隊長核心安全嗎？
- 能量條是什麼？

建議預告提示：

> 接下來，我們來看畫面下方的 HUD。
>
> 它會告訴你 Lattice 的生命、核心狀態，以及之後能使用的行動資源。

## 3. 教學第三段：HUD、隊長血條與攻擊計時

### 3.1 聚光燈放在哪裡

當玩家開始與敵人重疊，也就是 Lattice 進入攻擊範圍時，教學的聚光燈要從戰場中央轉到下方 HUD。

第一個要看的不是全部 HUD，而是兩個最關鍵的位置：

- 隊長血條。
- 隊長下方的攻擊計時條。

建議提示：

> 進入攻擊範圍後，看向下方 HUD。
>
> 這裡會顯示隊長的生命，以及我方下一次攻擊的計時。

這一段要讓玩家理解：「戰鬥不是只看圖騰碰撞，也要看 HUD 讀秒。」

### 3.2 隊長血條

隊長血條代表核心是否安全。它應該是教學中第一個被聚光燈框住的 HUD 元素。

建議提示：

> 這是隊長的生命。
>
> 隊長是 Lattice 的核心。請不要讓這條血量歸零。

這裡先不講完整隊員分攤、護盾或治療，只讓玩家建立一件事：

`隊長血條是最重要的失敗警報。`

### 3.2.1 為什麼血條在左邊，且隊長血條要獨立

HUD 的核心構圖是：

`左側血條 - 中央隊長頭像 - 右側能量條`

教學中可以直接解釋這個設計：

> 左邊是隊長生命。
>
> 隊長血條歸零，就代表整個 Lattice 瓦解。
>
> 所以隊長生命會獨立顯示，不混在隊員生命裡。

這樣玩家會知道：下方三個隊員生命欄位很重要，但它們不是失敗條件本身。真正不能歸零的是隊長血條。

### 3.3 攻擊計時條

隊長血條下面的計時條，用來表示我方攻擊頻率。

建議提示：

> 下方的計時條代表我方攻擊節奏。
>
> 只有進入攻擊範圍時，計時條才會開始前進。
>
> 當計時條填滿，Lattice 會完成一次攻擊。

攻擊完成時，畫面應該給玩家一個清楚但不吵的回饋：

- 計時條發出短暫亮光。
- 接觸中的 Lattice 層位或攻擊方向閃一下。
- 敵人受到傷害後，計時條立刻歸零並開始下一輪攻擊計時。

建議提示：

> 每次攻擊完成時，計時條會亮起，然後開始下一輪計時。

設計備註：

- 這條是「我方攻擊頻率」的教學，不要寫成敵我雙方必定同時攻擊。
- 之後敵人的攻擊頻率可以有自己的節奏、讀秒或觸發條件。
- 教學可以先讓玩家看懂我方計時條，再在後續敵人教學中補上「敵人也有自己的攻擊節奏」。

### 3.4 右側能量條

右邊的能量條代表隊伍目前可用的行動資源。

教學中要把它和生命條區隔清楚：

> 右邊是能量。
>
> 除了碰撞攻擊之外，許多主動行動都需要消耗能量。
>
> 能量不足時，你無法順利使用隊長技能或其他戰術動作。

這裡要讓玩家理解「生命決定能不能活下來，能量決定能不能做事」。如果只看生命、不看能量，玩家會以為自己安全；但其實可能已經失去逃跑或操作空間。

設計備註：日後可加入「使用藥水也需要少量能量」的規則。原因不是要用硬時間冷卻限制玩家，而是避免玩家在生命藥水或能量藥水很多時，無成本連續刷補給。用能量作為喝藥成本，可以形成柔性的冷卻感：玩家仍可連續操作，但必須承擔能量管理壓力。

### 3.5 隊長技能與 Space 鍵

在玩家看懂能量條後，可以提示玩家使用隊長技能。

目前預設隊長通常是 **Conductor**，可用它來做第一版教學：

> 按下 Space 使用隊長技能。
>
> Conductor 的技能會讓 Lattice 短暫加速，用來切入、拉開距離或重新調整位置。

隊長頭像外圍的一圈代表隊長技能冷卻狀態。

建議提示：

> 看隊長外圍的圓環。
>
> 圓環恢復後，代表隊長技能又準備好了。

這段教學要把三件事串起來：

- Space 會使用隊長技能。
- 使用技能需要能量。
- 隊長外圈顯示技能冷卻。

### 3.6 隊員生命欄位

玩家理解隊長血條與攻擊計時後，再看下面三個隊員欄位。

這三個欄位用來顯示目前選中層位中的隊員生命值。玩家可以透過左右切換外層、中層、內層。

建議提示：

> 下方三個欄位顯示隊員生命值。
>
> 你可以左右切換外層、中層、內層，查看不同層位的狀態。
>
> 盡量讓隊員生命保持健康，受傷太重的層位會讓 Lattice 變得危險。

這一段的目標不是教玩家立刻治療，而是讓玩家開始把「碰撞位置」和「隊員生命」連在一起。

設計備註：目前隊員欄位多以職責顏色（Defense / Firepower / Supply）進行編排。日後可以新增「排列方式切換」，讓玩家在以下兩種讀法間切換：

- **依職責排列**：看防守、火力、補給各自狀態。
- **依層位排列**：看內層、中層、外層各自狀態。

這個切換很適合放在隊員生命欄位附近，因為玩家受傷時有時想知道「哪個職責快倒了」，有時想知道「哪一層快破了」。

### 3.7 建議互動節奏

第三段可以拆成 7 個短任務：

1. **進入攻擊範圍**：玩家與黑白敵人重疊。
2. **聚光隊長血條**：提示隊長生命是核心安全指標。
3. **聚光攻擊計時條**：提示只有在攻擊範圍內才會計時。
4. **等待一次攻擊完成**：計時條滿、發光、攻擊、歸零。
5. **聚光右側能量條**：提示能量是主動行動資源。
6. **按 Space 使用 Conductor 技能**：讓玩家感受加速，並看見隊長外圈冷卻。
7. **查看隊員生命欄位**：提示可左右切換外/中/內層，保持隊員生命健康。

### 3.8 下一段預告：隊長核心與生存

HUD 教完後，下一段應該補強「為什麼隊長核心不能死」，因為玩家已經看過血條和攻擊計時，接下來自然要理解失敗條件。

建議預告提示：

> 接下來，要理解 Lattice 最重要的規則：隊長是核心。
>
> 只要核心倒下，整個 Lattice 就會崩解。

---

## 4. 教學第四段：補給物品與拾取掉落

### 4.1 右下角補給物品

玩家經過第一次戰鬥後，此時應該已經受傷。這是介紹補給物品的自然時機。

教學聚光燈可以移到右下角補給物品欄。

建議提示：

> 你受傷了。
>
> 右下角有目前可用的補給物品。
>
> 按下一次即可使用補給，但會消耗少量能量。

這裡要延續前面「能量是主動行動資源」的設計，讓玩家知道喝藥不是完全免費的動作。

補給物品的第一印象應該是：

- 使用一次會消耗能量。
- 使用後狀態會大幅提升。
- 補給是戰鬥中的救急手段，不是可以無限制亂刷的背景資源。

建議提示：

> 補給能大幅恢復狀態。
>
> 但使用補給也需要能量，請不要等能量完全見底才想自救。

### 4.2 補給物品從哪裡來

玩家第一次使用補給後，可以順手告訴他補給來源。

建議提示：

> 這些補給物品可以在世界各處的商店購買。
>
> 探索時看到商店，記得補充生命藥水與能量藥水。

這段不需要教完整商店 UI，也不需要講價格、刷新、世界特色藥水。先讓玩家知道「商店 = 補給來源」即可。

### 4.3 腳下零件與 E 鍵拾取

敵人被擊敗或資源物件被破壞後，玩家腳下附近可能會出現零件、素材或原石。

教學可以把聚光燈轉到 Lattice 腳下的掉落物。

建議提示：

> 留意腳下散落的零件。
>
> 靠近後按 E，可以拾取附近資源。

如果畫面有拾取清單，第一次要讓玩家看懂清單大致分成幾類：

- **綠色素材**：包含升級所需素材，以及少量原石。
- **原石**：通用資源，可用於商店交易與部分系統消耗。
- **其他零件/材料**：之後可用於升級、合成或交易。

建議提示：

> 綠色掉落通常包含升級素材與少量原石。
>
> 這些資源會讓你的 Lattice 在之後變得更強。

這裡先不教完整配方，只建立「拾取 = 成長資源」的直覺。

### 4.4 建議互動節奏

第四段可以拆成 4 個短任務：

1. **受傷後聚光右下角補給**：提示玩家目前有可用補給。
2. **使用一次補給**：補給消耗少量能量，但大幅恢復狀態。
3. **說明商店來源**：補給物品可在世界各處商店購買。
4. **按 E 拾取掉落**：玩家拾取腳下零件，看到綠色素材與少量原石。

### 4.5 下一段預告：物品欄與升級資源

補給與拾取教完後，玩家已經拿到資源。下一步應該帶玩家打開物品欄，看懂剛剛撿到的東西到底能做什麼。

建議預告提示：

> 補給能救急，資源能讓 Lattice 成長。
>
> 接下來，來看看這些素材會把哪一位隊員推向下一個星級。

## 5. 教學第五段：物品欄、升級素材與怪物分級

### 5.1 進入物品欄

玩家完成第一次拾取後，教學可以引導玩家打開物品欄。

這裡的目的不是完整教完所有養成，而是讓玩家理解三件事：

- 剛剛撿到的素材會進背包。
- 素材可以拿來讓隊員升星。
- 怪物等級會決定掉落素材的等級。

建議提示：

> 你剛剛取得的素材已經放進物品欄。
>
> 這裡可以查看目前擁有的原石、素材與補給。

教學用的背包可以預先放入「足夠讓一名隊員升一級」的資源。這次先不指定是哪一位隊員，避免玩家把注意力放在角色選擇，而不是資源規則。

建議提示：

> 目前的素材已經足夠讓一名隊員升一級。
>
> 先不用急著決定升誰，先看懂素材等級與用途。

### 5.2 素材等級與升星用途

物品欄中可以把素材依照用途分成三個層級：

- **0 -> 1 星解鎖素材**：讓尚未成形的隊員完成初始化，成為可以加入 Lattice 的單位。
- **1 -> 2 星升星素材**：讓已經是 1 星的隊員提升到 2 星，獲得更高的基礎能力。
- **2 -> 3 星升星素材**：讓 2 星隊員邁向 3 星，準備進入質變分支。

建議提示：

> 素材不是全部混在一起使用。
>
> 不同星級的隊員，需要對應階段的素材才能繼續成長。

這裡可以用一個小表格呈現：

| 素材用途 | 對應成長 | 教學說法 |
| --- | --- | --- |
| 0 -> 1 星 | 解鎖/初始化隊員 | 讓新的隊員真正加入 Lattice。 |
| 1 -> 2 星 | 初階升星 | 讓已加入的隊員變得更穩、更耐打、更有輸出。 |
| 2 -> 3 星 | 高階升星 | 讓隊員接近質變，之後可引出分支選擇。 |

目前系統文件中的角色升星方向是：角色升星本身不額外消耗原石，主要消耗對應世界的生物材料與家族碎片。較完整的配方採用 `5-3-1` 的比例，也就是：

- 1 個本階段最稀有/高級素材。
- 3 個本階段普通素材。
- 5 個上一階段高級素材。

教學中不必第一次就把 `5-3-1` 講成壓力，而可以說成「配方會顯示你還差哪幾種材料」。

建議提示：

> 升級面板會列出需要的素材數量。
>
> 只要材料足夠，就能選擇一名隊員進行升星。

### 5.3 原石的功能與來源

原石是通用資源，不是隊員升星的主要材料，但會在很多系統中被消耗。

它目前適合在教學中被描述成：

- 用來購買商店補給。
- 用於交易或部分系統消耗。
- 用於家族武器/技能升級等後續養成。
- 死亡復活時可能會損失一部分，因此不是完全沒有風險的貨幣。

建議提示：

> 原石是最通用的資源。
>
> 你可以用它購買補給、進行交易，之後也會用在技能與武器強化上。

原石的獲得渠道可以先介紹最基本的幾種：

- 擊敗怪物時獲得少量原石。
- 拾取掉落物時一併取得。
- 探索地圖資源或寶箱取得。
- 將多餘素材賣給商店換取原石。
- 部分資源型隊員日後可以提高原石產出。

這裡不要急著介紹全部價格與商店系統，先讓玩家知道「原石 = 通用成長貨幣」即可。

### 5.4 怪物分級與素材掉落

玩家剛剛擊敗的是 **T1 怪物**。這是介紹怪物分級的好時機。

建議提示：

> 剛剛擊敗的是 T1 怪物。
>
> T1 怪物通常提供低階素材與少量原石。

可以用非常短的分級說明建立直覺：

| 怪物階級 | 教學定位 | 主要掉落 |
| --- | --- | --- |
| T0 | 非攻擊型資源生命體 | 各階段基礎材料與極少量原石。 |
| T1 | 初期攻擊型弱小兵 | 0 -> 1 星素材與少量原石。 |
| T2 | 精英/中階敵人 | 1 -> 2 星素材與中等原石。 |
| T3 | 強襲小 Boss | 2 -> 3 星普通素材、大量 1 -> 2 星素材與大量原石。 |
| T4 | 世界 Boss | 2 -> 3 星關鍵核心素材、長期增益與極大量原石。 |

教學口吻可以是：

> 怪物越高階，掉落的素材也越高階。
>
> 想讓隊員繼續升星，就需要挑戰對應階級的怪物。

這樣玩家會自然得到一條成長邏輯：

1. 打 T1，取得 0 -> 1 星素材。
2. 打 T2，取得 1 -> 2 星素材。
3. 打 T3/T4，取得 2 -> 3 星素材與更重要的核心資源。

### 5.5 建議互動節奏

第五段可以拆成 5 個短任務：

1. **打開物品欄**：讓玩家看到剛剛拾取的素材與原石。
2. **指出可升級資源**：提示目前素材足以讓一名隊員升一級，但先不指定升誰。
3. **介紹素材用途**：說明 0 -> 1、1 -> 2、2 -> 3 分別對應不同成長階段。
4. **介紹原石**：說明它是補給、交易、技能與武器強化會使用的通用資源。
5. **介紹 T1 掉落**：說明剛剛擊敗的是 T1，因此掉落低階素材；更高階怪物會提供更高階素材。

### 5.6 下一段預告：隊長核心與生存

物品欄與素材用途講完後，再回到戰鬥生存。玩家此時已經知道資源能讓隊伍變強，接著要理解：所有成長都建立在隊長核心還活著的前提上。

建議預告提示：

> 素材能讓隊員升星，原石能支撐補給與強化。
>
> 但只要隊長核心倒下，這一次探索就會被迫中斷。

## 6. 第一段教學的互動節奏

建議第一段不要一次跳出大段文字，而是分成 4 個短提示：

1. **命名**：這個旋轉圖騰叫 Lattice。
2. **核心**：中央隊長是核心，核心倒下就失敗。
3. **三名初始隊員**：三個人是起始隊伍，位置代表不同戰術意義。
4. **成長承諾**：升級與招募會改變 Lattice 的圖紋與力量。

每一段都應該對應畫面上的高亮區域：

- 講 Lattice 時，高亮整個旋轉圓。
- 講隊長時，高亮中央核心。
- 講三名隊員時，高亮三個初始位置。
- 講圖紋成長時，可以讓 Lattice 做一次短暫的光紋脈動。

## 7. 暫定教學目標

第一段教學完成後，玩家應該知道：

- Lattice 是自己的隊伍圖騰。
- Lattice 的中央是隊長核心。
- 初始三人不是裝飾，而是起始隊伍。
- 升級會讓隊員圖紋改變。
- 招募新隊員可以強化並改變 Lattice。

第二段教學完成後，玩家應該知道：

- WASD 或方向鍵可以移動 Lattice。
- 黑白生命體是敵人。
- 敵人是這個世界自然產生的排列構造。
- Lattice 某一層碰觸敵人時，代表該層進入攻擊範圍。
- 外層接敵比讓敵人碰到核心安全。

第三段教學完成後，玩家應該知道：

- 隊長血條是核心安全指標。
- 左側血條獨立顯示，因為隊長歸零代表整隊瓦解。
- 攻擊計時條代表我方攻擊頻率。
- 只有進入攻擊範圍時，攻擊計時條才會前進。
- 計時條滿時會完成一次攻擊，亮光後開始下一輪計時。
- 右側能量條代表主動行動資源。
- Space 可以使用隊長技能。
- Conductor 的隊長技能可讓 Lattice 短暫加速。
- 隊長外圍圓環代表技能冷卻。
- 下方三個欄位可以查看隊員生命值。
- 玩家可以左右切換外層、中層、內層。
- 隊員生命要盡量保持健康。

第四段教學完成後，玩家應該知道：

- 右下角有補給物品。
- 使用補給會消耗少量能量。
- 補給可大幅恢復狀態。
- 補給物品可在世界各處商店購買。
- 腳下零件或掉落物可用 E 拾取。
- 綠色掉落通常包含升級素材與少量原石。
- 拾取資源是讓 Lattice 成長的基礎。

第五段教學完成後，玩家應該知道：

- 拾取到的素材與原石會進入物品欄。
- 教學預置資源足以讓一名隊員升一級，但不必立刻決定升誰。
- 0 -> 1 星、1 -> 2 星、2 -> 3 星需要對應階段的素材。
- 隊員升星主要消耗生物材料與家族碎片，角色升星本身不額外消耗原石。
- 原石是通用資源，可用於購買補給、交易、技能與武器強化等後續系統。
- T1 怪物會提供低階素材與少量原石。
- 怪物階級越高，通常提供越高階的升星素材。

第一段教學不需要玩家完全理解：

- 完整職責倍率。
- 所有星級分支。
- 所有武器家族與紅星。
- Boss、材料、商店與合成台。

那些應該留到後面的教學段落，否則第一口就太大，玩家會噎到。這遊戲已經夠有料了，教學不要一上來把料整鍋倒進碗裡。

## 9. Tutorial 任務式通關路線草案

這一段先把 Tutorial 規劃成「完整通關任務鏈」。它不是最終任務數量，而是後續設計每段任務時的骨架。

Tutorial 的任務可以分成三種：

- **主線任務**：必須完成，推動玩家走向通關。
- **支線任務**：用來教補給、商店、背包、素材、招募、換層等系統。
- **保護任務**：當玩家快死、迷路、資源不足時，系統用提示或補償把玩家拉回路線。

### 9.1 階段 0：自動開始與安全開場

**任務目的**：讓玩家在不做選擇的情況下進入 Tutorial，降低第一次遊玩的心理負擔。

任務狀態：

- 第一次遊玩且沒有遊玩紀錄時，自動開啟 Tutorial。
- 預設從 No Starting Boost 開始。
- 教學難度啟用：敵人壓力降低、初始資源可被系統補足、關鍵目標可被標記。

建議任務文字：

> 任務開始：觀察你的 Lattice。

完成條件：

- 玩家進入正式遊玩畫面。
- 第一段 Lattice 介紹完成。

### 9.2 階段 1：認識 Lattice

**任務目的**：讓玩家知道自己操作的不是單一角色，而是一個會成長的小隊圖騰。

建議任務：

1. 觀察中央旋轉的 Lattice。
2. 認識中央隊長核心。
3. 認識三名初始隊員。
4. 看一次圖紋脈動，理解升級會改變外觀。

完成條件：

- 玩家讀完或略過第一段提示。
- 高亮 Lattice、隊長、三名隊員各一次。

### 9.3 階段 2：第一次移動

**任務目的**：給玩家基本控制感。

建議任務：

1. 使用 WASD 或方向鍵移動。
2. 移動到地圖上標記的安全點。
3. 跟隨箭頭或地面標記前往第一個接敵區。

建議任務文字：

> 移動到光標標記的位置。

完成條件：

- 玩家移動超過一小段距離。
- 玩家抵達指定區域。

### 9.4 階段 3：第一次接近敵人

**任務目的**：讓玩家知道黑白生命體是敵人，並理解接觸範圍與攻擊範圍。

建議任務：

1. 接近一名 T1 敵人。
2. 讓外層隊員進入敵人攻擊範圍。
3. 觀察攻擊計時條開始累積。
4. 等待一次攻擊完成。

建議任務文字：

> 接近黑白敵人，讓外層隊員進入攻擊範圍。

完成條件：

- 玩家與 T1 敵人重疊或進入攻擊範圍。
- 我方攻擊計時條完成一次。
- 敵人受到一次傷害。

Tutorial 保護：

- 第一名敵人的傷害應該非常低。
- 如果玩家核心直接接敵，可以提示「試著讓外層先接觸敵人」。

### 9.5 階段 4：HUD 與隊伍狀態

**任務目的**：讓玩家在戰鬥中看懂血量、能量、攻擊節奏與隊員欄位。

建議任務：

1. 觀察隊長血條。
2. 觀察攻擊計時條。
3. 觀察能量條。
4. 按 Space 使用 Conductor 加速技能。
5. 使用左右切換外層、中層、內層隊員欄位。

完成條件：

- 玩家看到一次攻擊亮光。
- 玩家使用一次隊長技能。
- 玩家切換至少一次層位查看。

### 9.6 階段 5：補給與拾取

**任務目的**：把受傷、補給、能量消耗、掉落拾取串起來。

建議任務：

1. 玩家受傷後，提示右下角補給物品。
2. 使用一次補給。
3. 擊敗或完成第一名 T1 敵人。
4. 按 E 拾取腳下掉落物。

完成條件：

- 補給使用成功。
- 能量因補給而消耗。
- 掉落物被拾取。

Tutorial 保護：

- 若玩家沒有受傷，可以讓敵人造成極低安全傷害，或直接讓系統提示「你可以先試用一次補給」。
- 若玩家忘記拾取，掉落物可以持續發光並顯示 E 鍵提示。

### 9.7 階段 6：物品欄與第一次升級

**任務目的**：讓玩家理解撿到的素材不是裝飾，而是推動 Lattice 成長的資源。

建議任務：

1. 打開物品欄。
2. 查看剛剛拾取的素材與原石。
3. 查看可升級隊員。
4. 選擇一名系統推薦隊員升一級。
5. 觀察 Lattice 圖紋變化。

完成條件：

- 玩家打開物品欄。
- 至少完成一次隊員升級。
- 升級後播放一次 Lattice 圖紋變化。

設計備註：

- 目前先不決定推薦哪一位隊員。
- Tutorial 可以預先補足升一級所需資源，避免玩家卡在材料不足。

### 9.8 階段 7：怪物分級與下一個目標

**任務目的**：把「打什麼怪，拿什麼素材」變成玩家可以理解的路線。

建議任務：

1. 回顧剛剛擊敗的是 T1。
2. 告訴玩家 T1 提供低階素材。
3. 標記一個 T2 方向或下一個資源點。
4. 引導玩家前往下一個成長目標。

完成條件：

- 玩家理解 T1/T2/T3/T4 的基本掉落差異。
- 玩家接受下一個目標標記。

### 9.9 階段 8：隊長核心與失敗條件

**任務目的**：正式告訴玩家隊長血條歸零代表整個 Lattice 瓦解。

建議任務：

1. 高亮隊長血條。
2. 高亮中央核心。
3. 讓玩家看到安全示範：外層受傷與核心受傷的差異。
4. 提示玩家保護核心是所有行動的前提。

完成條件：

- 玩家完成一次外層接敵而核心未直接受擊。
- 玩家讀完核心失敗條件提示。

### 9.10 階段 9：第一次招募或隊伍擴張

**任務目的**：讓玩家從「三人起始 Lattice」進入「可成長隊伍」。

建議任務：

1. 前往第一個招募/雕像/合成點。
2. 查看可加入的隊員。
3. 完成一次招募或初始化。
4. 將新隊員放入指定層位。

完成條件：

- 新隊員加入 Lattice。
- Lattice 外觀增加新圖紋或新節點。

### 9.11 階段 10：第一個支線循環

**任務目的**：讓玩家做一次小型自由探索，但仍被任務框住。

可以提供三個支線，讓玩家選一個或照順序完成：

1. 擊敗 2 名 T1 敵人。
2. 拾取 3 份素材。
3. 找到商店並購買 1 個補給。

完成條件：

- 任一支線完成即可推進主線。
- 全部完成則給一點額外原石或補給獎勵。

### 9.12 階段 11：T2 教學戰

**任務目的**：讓玩家理解更高階敵人提供更高階素材，也需要更穩定的隊伍狀態。

建議任務：

1. 前往被標記的 T2 敵人區。
2. 戰前檢查隊長血量、隊員血量、補給與能量。
3. 與 T2 敵人交戰。
4. 擊敗後拾取 1 -> 2 星素材。

完成條件：

- T2 敵人被擊敗。
- 玩家拾取 1 -> 2 星素材。
- 系統提示 T2 與 T1 掉落差異。

Tutorial 保護：

- T2 可以使用弱化版本。
- 若玩家失敗，可在附近復活並保留教學進度。

### 9.13 階段 12：工作台/合成台與第二次成長

**任務目的**：把素材帶回可使用的地點，完成一次更正式的成長操作。

建議任務：

1. 前往工作台或合成台。
2. 查看升級配方。
3. 使用剛剛獲得的素材完成一次 1 -> 2 成長，或完成一個系統指定的替代升級。
4. 觀察數值與 Lattice 圖紋變化。

完成條件：

- 完成一次工作台/合成台操作。
- 玩家理解「素材要到特定互動點才能轉換成成長」。

### 9.14 階段 13：Boss 前準備

**任務目的**：讓玩家第一次理解通關不是一直往前撞，而是要準備隊伍、補給與資源。

建議任務：

1. 補滿或購買指定補給。
2. 確認隊伍至少完成指定次數升級。
3. 前往 Boss 區入口。
4. 閱讀 Boss 前提示。

完成條件：

- 補給數量達標。
- 隊伍成長達標。
- 玩家抵達 Boss 前區域。

### 9.15 階段 14：Tutorial Boss 與通關

**任務目的**：完成第一次通關，讓玩家把整個循環串起來。

Tutorial Boss 不一定要等同正式難度 Boss。它可以是簡化版，用來讓玩家練習：

- 接近敵人。
- 看攻擊節奏。
- 使用技能。
- 使用補給。
- 保護核心。
- 擊敗敵人後拾取通關物。

建議任務：

1. 進入 Tutorial Boss 區。
2. 依照提示躲避或換層。
3. 使用一次隊長技能。
4. 使用一次補給。
5. 擊敗 Boss。
6. 拾取 COrebound 核心鑰匙或 Tutorial 專用通關物。
7. 觸發通關演出。

完成條件：

- Boss 被擊敗。
- 玩家拾取通關物。
- 顯示 Tutorial 通關結算。

### 9.16 階段 15：通關後解鎖

**任務目的**：讓玩家知道自己已經完成教學，下一次可以選擇是否開啟教學。

建議結果：

- 標記玩家已有遊玩紀錄。
- New Game 顯示「需要教學嗎？」切換。
- 預設關閉教學。
- 顯示「你已完成 Tutorial，接下來可以嘗試 Easy 或 Normal」。

建議提示：

> 你已完成第一次 Lattice 通關。
>
> 下一次開始新遊戲時，你可以選擇是否再次開啟教學。
