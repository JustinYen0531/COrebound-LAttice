# 第一次遊玩教學流程草稿

本文件用來記錄第一次遊玩時的教學流程討論。它先不追求完整實作規格，而是把玩家第一次接觸遊戲時，應該先理解的畫面元素、用語、敘事順序與 UI 引導目標整理下來。

目前已定方向：

- 第一次沒有遊玩紀錄時，New Game 會自動開啟教學模式。
- 教學模式預設開場方式為 No Starting Boost。
- 玩家仍可在準備頁改選 Showcase。
- 玩家正式進場一次後，之後 New Game 才顯示「需要教學嗎？」切換。
- Tutorial 不是一般難度，而是比 Easy 更低的完整引導難度。
- Tutorial 的目標是手把手帶玩家完成一次完整通關，而不是測試玩家技術。
- Tutorial 會以一連串支線任務/主線任務的方式推進，讓玩家照著任務逐步跑完整個遊戲循環。

---

## 0. Tutorial 的整體定位：最低難度的任務式通關

Tutorial 應該被視為一種特殊難度層，而不是單純的新手提示。

如果之後遊戲有難度選擇，概念可以是：

| 難度 | 定位 |
| --- | --- |
| Tutorial | 比 Easy 更低。手把手帶玩家完成一次完整通關。 |
| Easy | 給已理解基本操作、但還不想承受太大壓力的玩家。 |
| Normal | 預設完整體驗，要求玩家自行規劃資源與戰鬥路線。 |
| Hard 以上 | 提高敵人壓力、資源壓力、Boss 機制與通關路線要求。 |

Tutorial 的核心不是「把所有規則講完」，而是讓玩家在幾乎不需要高技術的情況下，完成一次從開場到通關的完整流程。

它更像任務導覽：

- 玩家照著任務走，就能知道下一步該做什麼。
- 每一個任務只教一個主要概念。
- 必要時系統可以降低敵人壓力、補足資源、標記目標位置，甚至暫時限制玩家走錯方向。
- 教學通關後，玩家才真正理解普通難度裡「自己規劃路線」是什麼意思。

### 0.1 Tutorial 的任務設計原則

每個階段都應該符合以下節奏：

1. **先給目標**：玩家知道現在要做什麼。
2. **再給操作**：玩家知道要按什麼、去哪裡、碰什麼。
3. **完成後解釋意義**：玩家知道自己剛剛學到哪個系統。
4. **最後預告下一步**：玩家知道下一個任務為什麼自然接上來。

任務不要一次教太多。Tutorial 應該像有人牽著玩家走完第一趟旅程，而不是把一本說明書貼在玩家臉上。這點很關鍵，因為本作的系統密度很高；教學如果太像百科，玩家會被知識量壓扁。

## 1. 教學第一段：映入眼簾的旋轉圓圈

### 1.1 玩家第一眼要被告知什麼

玩家進入遊戲後，第一個需要理解的不是按鍵，也不是數值，而是畫面中央那個正在旋轉的圓形圖騰。

這個圖騰在世界觀裡稱為 **Lattice**。

Lattice 不是單純的裝飾，也不是普通血條。它同時代表：

- 玩家目前的小隊本體。
- 隊長與隊員之間的陣型關係。
- 隊伍升級、招募與成長後留下的視覺痕跡。
- 玩家在這個世界中的存在方式。

教學第一段要讓玩家知道：「我不是在操作一個單人角色，我是在培養、移動、保護一個會成長的 Lattice。」

### 1.2 建議教學文案草稿

> 你眼前旋轉的圓形圖騰，稱為 Lattice。
>
> Lattice 是你的小隊在這個世界中的形狀。它會隨著隊員、陣型與升級逐漸改變。
>
> 中央是隊長，也是整個 Lattice 的核心。只要核心倒下，這一局就會結束。
>
> 圍繞核心的三名初始隊員，是你第一層小隊力量。他們負責保護、輸出與支援，並且會在之後透過升級與招募，讓整個 Lattice 變得更完整。

### 1.3 這三個人代表什麼

第一次教學中，畫面上的三名初始隊員要被解釋成「Lattice 的第一個骨架」，而不是單純三張角色卡。

三名初始隊員可以用以下方式介紹：

- **內層隊員**：靠近隊長，較安全，代表保護核心與穩定起步。
- **中層隊員**：承接內外層，代表隊伍的平衡與轉換空間。
- **外層隊員**：最接近外部威脅，代表第一線接觸、輸出與承傷。

這一段不需要一開始就講完整職責倍率。第一次玩家還在認圖，不要立刻丟數字。先讓他理解「位置有意義」就好。

### 1.4 升級時 Lattice 會怎麼變

玩家需要被告知：每次升級不只是數值變強，角色所屬的圖紋也會變化。

建議用語：

- 每位隊員都有自己的圖紋。
- 隊員升級後，圖紋會變得更明顯、更複雜。
- Lattice 會把這些圖紋編織進整體外觀。
- 因此玩家的隊伍不是只在數字上成長，也會在畫面上留下自己的形狀。

這是 COrebound LAttence 很重要的第一印象：成長不是只看面板，而是「你養出的圖騰真的變了」。

### 1.5 招募其他隊員的意義

玩家也要在第一段知道，Lattice 不是固定三個人。

教學可以這樣接：

> 你現在只看見 Lattice 的起始形狀。
>
> 在探索世界時，你可以招募其他隊員，讓他們加入不同層位。新的隊員會帶來新的能力，也會把新的圖紋加入 Lattice。
>
> 隊伍越完整，Lattice 就越強，也越接近你自己的戰鬥風格。

這一段的目的不是立刻教玩家怎麼招募，而是先埋下動機：「我等一下探索世界，是為了讓這個圖騰變完整。」

## 2. 教學第二段：移動與第一次碰觸敵人

### 2.1 玩家要做的第一個動作

介紹完 Lattice 之後，玩家需要立刻獲得控制感。

這一段的教學目標很單純：讓玩家知道 Lattice 會作為一整個小隊一起移動。

建議提示：

> 使用 WASD 或方向鍵移動 Lattice。
>
> 你不是在移動單一角色，而是在移動整個小隊圖騰。

如果要做成 UI 任務，可以設為：

- 任務文字：`使用 WASD 或方向鍵移動`
- 完成條件：玩家任意方向移動超過短距離。
- 完成回饋：Lattice 外圈輕微亮起，提示玩家「整個圖騰都被你帶動了」。

### 2.2 引導玩家碰觸黑白敵人

移動完成後，下一個教學目標是讓玩家主動接觸第一個黑白敵人。

這裡的提示不要急著把整套世界觀全部講出來。第一次只要讓玩家知道：

- 黑白的是敵人。
- 它們是這個世界自然產生的生命體。
- 它們象徵一切連結中排列組合的構造。
- 玩家可以用 Lattice 的外層去碰觸它們，理解碰撞戰鬥的基本規則。

建議提示：

> 前方黑白的生命體是敵人。
>
> 先靠近它，讓 Lattice 的外層碰到它。

可選的世界觀補充，不建議第一時間全顯示：

> 這些生命體是世界自然生成的排列構造。它們不是普通野獸，而是連結、圖紋與規則在世界裡長出的形狀。

COLA 的部分此時只留在設計文件備註中，不在遊戲內第一段明講：

- 一切排列組合的極大成，最終會指向 COLA。
- 但第一次戰鬥教學不應直接說明 COLA，否則會過早暴露終局概念，也會讓玩家在還沒學會走路時先背世界史。

### 2.3 第一次進入攻擊範圍的規則

玩家碰到黑白敵人時，要先解釋最基礎的攻擊範圍概念：

> 當 Lattice 的某一層與敵人重疊時，就代表進入攻擊範圍。

更白話的提示版本：

> 外層碰到敵人時，外層正在承受風險，也會開始準備攻擊。
>
> 用外層接敵，比讓敵人碰到核心安全。

這段教學先只教「進入攻擊範圍」。不要在這一刻同時教職責倍率、護盾、治療、控制效果。玩家只要先建立一個本能：

`讓外層去碰敵人，不要讓敵人碰核心。`

設計備註：之後我方與敵方的攻擊頻率應該可以拆開，不必設計成雙方同時受傷或同時攻擊。教學文字要避免把「同時互傷」說死，而是先用「進入攻擊範圍」和「我方攻擊頻率」來承接。

### 2.4 建議互動節奏

第二段可以拆成 4 個短任務：

1. **移動 Lattice**：按 WASD 或方向鍵，讓玩家感覺整個圖騰被自己帶動。
2. **看見黑白敵人**：高亮前方敵人，文字只說「黑白的是敵人」。
3. **用外層碰觸敵人**：引導玩家用 Lattice 外層接觸敵人。
4. **理解攻擊範圍**：顯示外層進入攻擊範圍，接著把注意力導向 HUD 的攻擊計時條。

### 2.5 下一段預告：HUD

碰撞完成後，不要立刻進入複雜養成。下一段應該介紹 HUD，因為玩家剛剛已經看見自己受傷，心中自然會問：

- 我現在還剩多少生命？
- 哪一層受傷？
- 隊長核心安全嗎？
- 能量條是什麼？

建議預告提示：

> 接下來，我們來看畫面下方的 HUD。
>
> 它會告訴你 Lattice 的生命、核心狀態，以及之後能使用的行動資源。

## 3. 教學第三段：HUD、隊長血條與攻擊計時

### 3.1 聚光燈放在哪裡

當玩家開始與敵人重疊，也就是 Lattice 進入攻擊範圍時，教學的聚光燈要從戰場中央轉到下方 HUD。

第一個要看的不是全部 HUD，而是兩個最關鍵的位置：

- 隊長血條。
- 隊長下方的攻擊計時條。

建議提示：

> 進入攻擊範圍後，看向下方 HUD。
>
> 這裡會顯示隊長的生命，以及我方下一次攻擊的計時。

這一段要讓玩家理解：「戰鬥不是只看圖騰碰撞，也要看 HUD 讀秒。」

### 3.2 隊長血條

隊長血條代表核心是否安全。它應該是教學中第一個被聚光燈框住的 HUD 元素。

建議提示：

> 這是隊長的生命。
>
> 隊長是 Lattice 的核心。請不要讓這條血量歸零。

這裡先不講完整隊員分攤、護盾或治療，只讓玩家建立一件事：

`隊長血條是最重要的失敗警報。`

### 3.2.1 為什麼血條在左邊，且隊長血條要獨立

HUD 的核心構圖是：

`左側血條 - 中央隊長頭像 - 右側能量條`

教學中可以直接解釋這個設計：

> 左邊是隊長生命。
>
> 隊長血條歸零，就代表整個 Lattice 瓦解。
>
> 所以隊長生命會獨立顯示，不混在隊員生命裡。

這樣玩家會知道：下方三個隊員生命欄位很重要，但它們不是失敗條件本身。真正不能歸零的是隊長血條。

### 3.3 攻擊計時條

隊長血條下面的計時條，用來表示我方攻擊頻率。

建議提示：

> 下方的計時條代表我方攻擊節奏。
>
> 只有進入攻擊範圍時，計時條才會開始前進。
>
> 當計時條填滿，Lattice 會完成一次攻擊。

攻擊完成時，畫面應該給玩家一個清楚但不吵的回饋：

- 計時條發出短暫亮光。
- 接觸中的 Lattice 層位或攻擊方向閃一下。
- 敵人受到傷害後，計時條立刻歸零並開始下一輪攻擊計時。

建議提示：

> 每次攻擊完成時，計時條會亮起，然後開始下一輪計時。

設計備註：

- 這條是「我方攻擊頻率」的教學，不要寫成敵我雙方必定同時攻擊。
- 之後敵人的攻擊頻率可以有自己的節奏、讀秒或觸發條件。
- 教學可以先讓玩家看懂我方計時條，再在後續敵人教學中補上「敵人也有自己的攻擊節奏」。

### 3.4 右側能量條

右邊的能量條代表隊伍目前可用的行動資源。

教學中要把它和生命條區隔清楚：

> 右邊是能量。
>
> 除了碰撞攻擊之外，許多主動行動都需要消耗能量。
>
> 能量不足時，你無法順利使用隊長技能或其他戰術動作。

這裡要讓玩家理解「生命決定能不能活下來，能量決定能不能做事」。如果只看生命、不看能量，玩家會以為自己安全；但其實可能已經失去逃跑或操作空間。

設計備註：日後可加入「使用藥水也需要少量能量」的規則。原因不是要用硬時間冷卻限制玩家，而是避免玩家在生命藥水或能量藥水很多時，無成本連續刷補給。用能量作為喝藥成本，可以形成柔性的冷卻感：玩家仍可連續操作，但必須承擔能量管理壓力。

### 3.5 隊長技能與 Space 鍵

在玩家看懂能量條後，可以提示玩家使用隊長技能。

目前預設隊長通常是 **Conductor**，可用它來做第一版教學：

> 按下 Space 使用隊長技能。
>
> Conductor 的技能會讓 Lattice 短暫加速，用來切入、拉開距離或重新調整位置。

隊長頭像外圍的一圈代表隊長技能冷卻狀態。

建議提示：

> 看隊長外圍的圓環。
>
> 圓環恢復後，代表隊長技能又準備好了。

這段教學要把三件事串起來：

- Space 會使用隊長技能。
- 使用技能需要能量。
- 隊長外圈顯示技能冷卻。

### 3.6 隊員生命欄位

玩家理解隊長血條與攻擊計時後，再看下面三個隊員欄位。

這三個欄位用來顯示目前選中層位中的隊員生命值。玩家可以透過左右切換外層、中層、內層。

建議提示：

> 下方三個欄位顯示隊員生命值。
>
> 你可以左右切換外層、中層、內層，查看不同層位的狀態。
>
> 盡量讓隊員生命保持健康，受傷太重的層位會讓 Lattice 變得危險。

這一段的目標不是教玩家立刻治療，而是讓玩家開始把「碰撞位置」和「隊員生命」連在一起。

設計備註：目前隊員欄位多以職責顏色（Defense / Firepower / Supply）進行編排。日後可以新增「排列方式切換」，讓玩家在以下兩種讀法間切換：

- **依職責排列**：看防守、火力、補給各自狀態。
- **依層位排列**：看內層、中層、外層各自狀態。

這個切換很適合放在隊員生命欄位附近，因為玩家受傷時有時想知道「哪個職責快倒了」，有時想知道「哪一層快破了」。

### 3.7 建議互動節奏

第三段可以拆成 7 個短任務：

1. **進入攻擊範圍**：玩家與黑白敵人重疊。
2. **聚光隊長血條**：提示隊長生命是核心安全指標。
3. **聚光攻擊計時條**：提示只有在攻擊範圍內才會計時。
4. **等待一次攻擊完成**：計時條滿、發光、攻擊、歸零。
5. **聚光右側能量條**：提示能量是主動行動資源。
6. **按 Space 使用 Conductor 技能**：讓玩家感受加速，並看見隊長外圈冷卻。
7. **查看隊員生命欄位**：提示可左右切換外/中/內層，保持隊員生命健康。

### 3.8 下一段預告：補給與拾取

HUD 教完後，下一段應該先接補給與拾取，因為玩家已經看過血條、能量與攻擊計時，接下來自然要理解受傷後怎麼自救，以及擊敗敵人後如何取得資源。

建議預告提示：

> 你受傷了。
>
> 右下角有可以使用的補給。
>
> 擊敗敵人後，也別忘了拾取腳下的資源。

---

## 4. 教學第四段：補給物品與拾取掉落

### 4.1 右下角補給物品

玩家經過第一次戰鬥後，此時應該已經受傷。這是介紹補給物品的自然時機。

教學聚光燈可以移到右下角補給物品欄。

建議提示：

> 你受傷了。
>
> 右下角有目前可用的補給物品。
>
> 按下一次即可使用補給，但會消耗少量能量。

這裡要延續前面「能量是主動行動資源」的設計，讓玩家知道喝藥不是完全免費的動作。

補給物品的第一印象應該是：

- 使用一次會消耗能量。
- 使用後狀態會大幅提升。
- 補給是戰鬥中的救急手段，不是可以無限制亂刷的背景資源。

建議提示：

> 補給能大幅恢復狀態。
>
> 但使用補給也需要能量，請不要等能量完全見底才想自救。

### 4.2 補給物品從哪裡來

玩家第一次使用補給後，可以順手告訴他補給來源。

建議提示：

> 這些補給物品可以在世界各處的商店購買。
>
> 探索時看到商店，記得補充生命藥水與能量藥水。

這段不需要教完整商店 UI，也不需要講價格、刷新、世界特色藥水。先讓玩家知道「商店 = 補給來源」即可。

### 4.3 腳下零件與 E 鍵拾取

敵人被擊敗或資源物件被破壞後，玩家腳下附近可能會出現零件、素材或原石。

教學可以把聚光燈轉到 Lattice 腳下的掉落物。

建議提示：

> 留意腳下散落的零件。
>
> 靠近後按 E，可以拾取附近資源。

如果畫面有拾取清單，第一次要讓玩家看懂清單大致分成幾類：

- **綠色素材**：包含升級所需素材，以及少量原石。
- **原石**：通用資源，可用於商店交易與部分系統消耗。
- **其他零件/材料**：之後可用於升級、合成或交易。

建議提示：

> 綠色掉落通常包含升級素材與少量原石。
>
> 這些資源會讓你的 Lattice 在之後變得更強。

這裡先不教完整配方，只建立「拾取 = 成長資源」的直覺。

### 4.4 建議互動節奏

第四段可以拆成 4 個短任務：

1. **受傷後聚光右下角補給**：提示玩家目前有可用補給。
2. **使用一次補給**：補給消耗少量能量，但大幅恢復狀態。
3. **說明商店來源**：補給物品可在世界各處商店購買。
4. **按 E 拾取掉落**：玩家拾取腳下零件，看到綠色素材與少量原石。

### 4.5 下一段預告：物品欄與升級資源

補給與拾取教完後，玩家已經拿到資源。下一步應該帶玩家打開物品欄，看懂剛剛撿到的東西到底能做什麼。

建議預告提示：

> 補給能救急，資源能讓 Lattice 成長。
>
> 接下來，來看看這些素材會把哪一位隊員推向下一個星級。

## 5. 教學第五段：物品欄、升級素材與怪物分級

### 5.1 進入物品欄

玩家完成第一次拾取後，教學可以引導玩家打開物品欄。

這裡的目的不是完整教完所有養成，而是讓玩家理解三件事：

- 剛剛撿到的素材會進背包。
- 素材可以拿來讓隊員升星。
- 怪物等級會決定掉落素材的等級。

建議提示：

> 你剛剛取得的素材已經放進物品欄。
>
> 這裡可以查看目前擁有的原石、素材與補給。

教學用的背包可以預先放入「足夠讓一名隊員升一級」的資源。這次先不指定是哪一位隊員，避免玩家把注意力放在角色選擇，而不是資源規則。

建議提示：

> 目前的素材已經足夠讓一名隊員升一級。
>
> 先不用急著決定升誰，先看懂素材等級與用途。

### 5.2 素材等級與升星用途

物品欄中可以把素材依照用途分成三個層級：

- **0 -> 1 星解鎖素材**：讓尚未成形的隊員完成初始化，成為可以加入 Lattice 的單位。
- **1 -> 2 星升星素材**：讓已經是 1 星的隊員提升到 2 星，獲得更高的基礎能力。
- **2 -> 3 星升星素材**：讓 2 星隊員邁向 3 星，準備進入質變分支。

建議提示：

> 素材不是全部混在一起使用。
>
> 不同星級的隊員，需要對應階段的素材才能繼續成長。

這裡可以用一個小表格呈現：

| 素材用途 | 對應成長 | 教學說法 |
| --- | --- | --- |
| 0 -> 1 星 | 解鎖/初始化隊員 | 讓新的隊員真正加入 Lattice。 |
| 1 -> 2 星 | 初階升星 | 讓已加入的隊員變得更穩、更耐打、更有輸出。 |
| 2 -> 3 星 | 高階升星 | 讓隊員接近質變，之後可引出分支選擇。 |

目前系統文件中的角色升星方向是：角色升星本身不額外消耗原石，主要消耗對應世界的生物材料與家族碎片。較完整的配方採用 `5-3-1` 的比例，也就是：

- 1 個本階段最稀有/高級素材。
- 3 個本階段普通素材。
- 5 個上一階段高級素材。

教學中不必第一次就把 `5-3-1` 講成壓力，而可以說成「配方會顯示你還差哪幾種材料」。

建議提示：

> 升級面板會列出需要的素材數量。
>
> 只要材料足夠，就能選擇一名隊員進行升星。

### 5.3 原石的功能與來源

原石是通用資源，不是隊員升星的主要材料，但會在很多系統中被消耗。

它目前適合在教學中被描述成：

- 用來購買商店補給。
- 用於交易或部分系統消耗。
- 用於家族武器/技能升級等後續養成。
- 死亡復活時可能會損失一部分，因此不是完全沒有風險的貨幣。

建議提示：

> 原石是最通用的資源。
>
> 你可以用它購買補給、進行交易，之後也會用在技能與武器強化上。

原石的獲得渠道可以先介紹最基本的幾種：

- 擊敗怪物時獲得少量原石。
- 拾取掉落物時一併取得。
- 探索地圖資源或寶箱取得。
- 將多餘素材賣給商店換取原石。
- 部分資源型隊員日後可以提高原石產出。

這裡不要急著介紹全部價格與商店系統，先讓玩家知道「原石 = 通用成長貨幣」即可。

### 5.4 怪物分級與素材掉落

玩家剛剛擊敗的是 **T1 怪物**。這是介紹怪物分級的好時機。

建議提示：

> 剛剛擊敗的是 T1 怪物。
>
> T1 怪物通常提供低階素材與少量原石。

可以用非常短的分級說明建立直覺：

| 怪物階級 | 教學定位 | 主要掉落 |
| --- | --- | --- |
| T0 | 非攻擊型資源生命體 | 各階段基礎材料與極少量原石。 |
| T1 | 初期攻擊型弱小兵 | 0 -> 1 星素材與少量原石。 |
| T2 | 精英/中階敵人 | 1 -> 2 星素材與中等原石。 |
| T3 | 強襲小 Boss | 2 -> 3 星普通素材、大量 1 -> 2 星素材與大量原石。 |
| T4 | 世界 Boss | 2 -> 3 星關鍵核心素材、長期增益與極大量原石。 |

教學口吻可以是：

> 怪物越高階，掉落的素材也越高階。
>
> 想讓隊員繼續升星，就需要挑戰對應階級的怪物。

這樣玩家會自然得到一條成長邏輯：

1. 打 T1，取得 0 -> 1 星素材。
2. 打 T2，取得 1 -> 2 星素材。
3. 打 T3/T4，取得 2 -> 3 星素材與更重要的核心資源。

### 5.5 建議互動節奏

第五段可以拆成 5 個短任務：

1. **打開物品欄**：讓玩家看到剛剛拾取的素材與原石。
2. **指出可升級資源**：提示目前素材足以讓一名隊員升一級，但先不指定升誰。
3. **介紹素材用途**：說明 0 -> 1、1 -> 2、2 -> 3 分別對應不同成長階段。
4. **介紹原石**：說明它是補給、交易、技能與武器強化會使用的通用資源。
5. **介紹 T1 掉落**：說明剛剛擊敗的是 T1，因此掉落低階素材；更高階怪物會提供更高階素材。

### 5.6 下一段預告：隊長核心與生存

物品欄與素材用途講完後，再回到戰鬥生存。玩家此時已經知道資源能讓隊伍變強，接著要理解：所有成長都建立在隊長核心還活著的前提上。

建議預告提示：

> 素材能讓隊員升星，原石能支撐補給與強化。
>
> 但只要隊長核心倒下，這一次探索就會被迫中斷。

## 6. 教學第六段：隊長核心與失敗條件

### 6.1 這一段為什麼放在這裡

玩家已經知道素材、升級、原石與怪物分級，接下來要把注意力拉回最重要的生存規則：所有成長都建立在隊長核心還活著的前提上。

這段不是新增一個複雜系統，而是把前面零散提過的核心規則正式收束。

### 6.2 建議任務

1. 高亮隊長血條。
2. 高亮中央核心。
3. 讓玩家看到外層受傷與核心受傷的差異。
4. 提示玩家保護核心是所有行動的前提。

建議提示：

> 中央隊長是整個 Lattice 的核心。
>
> 隊員受傷時，隊伍還能調整；但核心倒下，這一次探索就會中斷。

### 6.3 完成條件

- 玩家完成一次外層接敵，且核心沒有直接受擊。
- 玩家讀完核心失敗條件提示。

### 6.4 下一段預告：隊伍擴張

> 保護核心不是只能靠閃避。
>
> 讓更多隊員加入 Lattice，才是長期生存的方式。

## 7. 教學第七段：第一次招募或隊伍擴張

### 7.1 這一段要教什麼

玩家已經知道 Lattice 會成長，這一段要讓他第一次真的看到「新的隊員加入」。

這裡可以用召喚/初始化雕像作為第一個建築接觸點，但重點不是完整講完所有建築，而是讓玩家理解：地圖上的互動物件會把素材轉換成隊伍成長。

### 7.2 建議任務

1. 跟隨標記前往第一個召喚/初始化雕像。
2. 查看可加入的隊員。
3. 使用 Tutorial 預置素材完成一次初始化，或查看初始化需求。
4. 將新隊員放入指定層位。
5. 觀察 Lattice 新增圖紋或新節點。

建議提示：

> 這座雕像能讓新的隊員加入 Lattice。

### 7.3 完成條件

- 玩家接觸第一座雕像。
- 新隊員加入 Lattice，或玩家看懂初始化需求。

### 7.4 下一段預告：長期素材任務

>
> 但之後的成長，仍會一路需要你擊敗敵人、拾取素材。

## 8. 教學第八段：長期素材任務與自由探索

### 8.1 這一段要解決的問題

正式難度中，玩家可能需要先擊敗敵人、累積素材，才有辦法操作某些建築。

但 Tutorial 不應該把玩家卡在刷材料上。因此這裡要採用兩層設計：

- **主線示範**：Tutorial 直接給一包基礎素材與原石，確保建築導覽能順利進行。
- **長期任務**：擊敗敵人、拾取素材、累積原石會一路掛在任務列表中，玩家邊走邊完成。

### 8.2 建議長期任務

1. 擊敗 2 名 T1 敵人。
2. 拾取 3 份素材。
3. 累積指定數量原石。
4. 拾取任意 1 份 1 -> 2 星素材。

建議任務文字：

> 長期任務：擊敗敵人並收集素材。
>
> 素材會一路累積，你不需要現在就刷滿所有材料。

### 8.3 Tutorial 直接補給

- 在建築導覽開始前，系統給玩家一組基礎素材與原石。
- 這包素材只用來確保每個早期建築都能展示一次核心功能。
- 玩家自己打到的材料仍然保留，並推進長期任務進度。

### 8.4 完成條件

- 長期任務不阻擋主線推進。
- 每完成一項，就給一個小獎勵或一段簡短說明。
- 全部完成時，給額外原石、補給或一份可用素材。

### 8.5 下一段預告：第一批建築

## 9. 教學第九段：第一批建築導覽

### 9.1 這一批建築的定位

第一批建築是玩家前期一定要早點接觸的建築。它們不需要玩家理解完整後期規劃，只要知道「這個建築能把某種資源轉換成某種成長」即可。

| 建築 | 教學定位 | Tutorial 介紹重點 |
| --- | --- | --- |
| 召喚/初始化雕像 | 讓新隊員加入 Lattice | 供奉素材，完成 0 -> 1 星初始化。 |

### 9.2 建議任務

這一段可以分為多個子任務，玩家可以依自己想探索的路線選擇順序。

## 10. 教學第十段：武器欄、能量消耗與 Auto Fire

### 10.1 這一段要教什麼

玩家在第一批建築中已經看過熔爐與合成台，這時候可以正式介紹「右側小黑框」代表什麼。

右側小黑框不是裝飾，而是目前 Lattice 可用的家族武器狀態。它會讓玩家知道：

- 目前有哪些武器家族正在啟用。
- 每個武器都有自己的發射節奏或冷卻。
- 武器發射會消耗共用能量。
- 可以用 Auto Fire / Hold Fire 控制是否自動發射。
- 必要時也可以關閉個別武器，避免能量被不想用的武器消耗掉。

### 10.2 五種武器家族的第一印象

這一段不用講完整數值，只建立「每一格代表一種武器傾向」。

| 武器 | 教學定位 | 第一次介紹說法 |
| --- | --- | --- |
| 護盾 | 防禦與推進 | 產生防禦型衝擊，幫 Lattice 擋住或壓開威脅。 |
| 多發 | 近距離爆發 | 一次打出多個彈體，適合在靠近敵人時提高壓力。 |
| 直線 | 穩定射擊 | 朝目標方向發射穩定直線攻擊，是最直覺的輸出方式。 |
| 地雷 | 區域控制 | 在場上留下危險區域，讓敵人靠近時付出代價。 |
| 激光 | 持續/高速打擊 | 以高速射線或持續判定造成壓力，之後可作為高階武器理解。 |

建議提示：

> 右側小黑框顯示你的家族武器。
>
> 護盾、多發、直線、地雷與激光，代表不同的攻擊方式。

### 10.3 能量消耗與武器冷卻

武器不是免費一直射。每次發射都會消耗右側能量條，並且每個武器都有自己的發射節奏或冷卻狀態。

建議提示：

> 武器會自動尋找時機發射。
>
> 但每次發射都會消耗能量，也需要等待自己的冷卻或發射節奏。
>
> 能量不足時，武器會暫停發射。

這裡要讓玩家理解兩層限制：

- **冷卻/發射節奏**：武器準備好了才會發射。
- **能量**：就算武器準備好了，能量不足也不能發射。

### 10.4 Auto Fire / Hold Fire

Auto Fire 是全隊武器的總開關。

- **Auto Fire 開啟**：武器在條件符合時會自動發射。
- **Hold Fire 開啟**：全隊武器暫停自動發射，保留能量。
- **個別武器關閉**：即使 Auto Fire 開啟，該武器也不會發射。

建議任務：

1. 高亮右側小黑框。
2. 顯示目前啟用的武器家族。
3. 讓玩家觀察一次武器冷卻或發射節奏。
4. 切換一次 Hold Fire，觀察武器停止自動發射。
5. 切回 Auto Fire，觀察武器恢復。
6. 關閉其中一個個別武器，再重新開啟。

建議提示：

> Auto Fire 會讓武器自動發射。
>
> 如果你想保留能量，可以切到 Hold Fire。
>
> 你也可以關閉某一種武器，只留下現在想使用的攻擊方式。

### 10.5 完成條件

- 玩家看過右側武器欄高亮。
- 玩家知道五種武器名稱。
- 玩家看過一次武器冷卻或發射狀態。
- 玩家切換過 Auto Fire / Hold Fire。
- 玩家至少關閉或開啟過一個個別武器。

### 10.6 下一段預告：T2 教學戰

> 你已經知道武器會消耗能量，也能控制是否自動發射。
>
> 接下來要挑戰 T2 敵人，練習在戰鬥前檢查血量、能量、補給與武器狀態。

## 11. 教學第十一段：T2 教學戰

### 11.1 這一段要教什麼

這一段用 T2 教學戰說明：更高階敵人提供更高階素材，也需要更穩定的隊伍狀態。

### 11.2 建議任務

1. 前往被標記的 T2 敵人區。
2. 戰前檢查隊長血量、隊員血量、補給、能量與武器 Auto Fire 狀態。
3. 與 T2 敵人交戰。
4. 擊敗後拾取 1 -> 2 星素材。

建議提示：

> T2 敵人比 T1 更危險，但也會提供更高階的素材。
>
> 戰鬥前先確認血量、能量、補給與武器狀態。

### 11.3 完成條件

- T2 敵人被擊敗。
- 玩家拾取 1 -> 2 星素材。
- 系統提示 T2 與 T1 掉落差異。

### 11.4 Tutorial 保護

- T2 可以使用弱化版本。
- 若玩家失敗，可在附近復活並保留教學進度。

### 11.5 下一段預告：第二批條件建築

> 你已經拿到更高階的素材。
>
> 有些建築會因為進度、素材或擊殺條件而開始變得重要。

## 12. 教學第十二段：第二批條件建築導覽

### 12.1 這一批建築的定位

第二批建築需要條件、進度或更高階資源才能完整使用。這批不要太早塞給玩家，否則前期教學會變成觀光導覽。

| 建築/互動點 | 解鎖條件 | 教學介紹重點 |
| --- | --- | --- |
| 高階合成台 | 取得 1 -> 2 或 2 -> 3 素材後 | 高階升星、技能合成或更長時間的合成鎖定。 |
| 專用家族熔爐 | 取得多餘材料或指定家族素材後 | 把材料轉成特定家族碎片，服務武器/技能養成。 |
| Boss 祭壇/召喚點 | 完成指定擊殺或取得印記後 | 不是補給點，而是推進世界 Boss 的入口。 |
| 傳送門/跨區入口 | 完成前置區域目標後 | 讓玩家進入更危險也更高收益的區域。 |
| 寶箱/特殊資源點 | 需要能量或特定條件 | 用能量換大量原石或隨機材料。 |

### 12.2 建議任務

1. 顯示「條件建築」標記，提醒玩家有些互動點現在只能查看，不能完整操作。
2. 前往一個已解鎖的第二批建築。
3. 查看它需要的條件與可提供的回報。
4. 完成一次條件已滿足的操作，或把它加入追蹤目標。

### 12.3 完成條件

- 玩家理解建築分為「早期可用」與「條件解鎖」兩批。
- 玩家至少查看 1 個第二批建築的解鎖條件。
- 系統把未解鎖建築加入可追蹤目標，而不是要求玩家立刻完成。

### 12.4 下一段預告：第二次正式成長

> 有些建築現在只是先讓你認路。
>
> 但合成台會立刻讓素材變成隊伍成長。

## 13. 教學第十三段：工作台/合成台與第二次成長

### 13.1 這一段要教什麼

這一段把素材帶回可使用的地點，完成一次更正式的成長操作。

### 13.2 建議任務

1. 前往工作台或合成台。
2. 查看升級配方。
3. 使用剛剛獲得的素材完成一次 1 -> 2 成長，或完成一個系統指定的替代升級。
4. 觀察數值與 Lattice 圖紋變化。

### 13.3 完成條件

- 完成一次工作台/合成台操作。
- 玩家理解「素材要到特定互動點才能轉換成成長」。

### 13.4 下一段預告：Boss 前準備

> Lattice 已經變得更完整。
>
> 接下來要準備補給、能量與隊伍狀態，前往第一次通關挑戰。

---

## 14. 教學第十四段：Boss 前準備

### 14.1 這一段要教什麼

通關不是一直往前撞，而是要準備隊伍、補給與資源。

### 14.2 建議任務

1. 補滿或購買指定補給。
2. 確認隊伍至少完成指定次數升級。
3. 確認 Auto Fire / Hold Fire 狀態與要啟用的武器。
4. 前往 Boss 區入口。
5. 閱讀 Boss 前提示。

### 14.3 完成條件

- 補給數量達標。
- 隊伍成長達標。
- 武器自動發射狀態已確認。
- 玩家抵達 Boss 前區域。

### 14.4 下一段預告：Tutorial Boss

> 這不是正式難度的考驗。
>
> 這是最後一次把你學過的操作串起來。

---

## 15. 教學第十五段：Tutorial Boss 與通關

### 15.1 這一段要教什麼

Tutorial Boss 不一定要等同正式難度 Boss。它可以是簡化版，用來讓玩家練習：

- 接近敵人。
- 看攻擊節奏。
- 使用技能。
- 使用補給。
- 保護核心。
- 擊敗敵人後拾取通關物。

### 15.2 建議任務

1. 進入 Tutorial Boss 區。
2. 依照提示躲避或換層。
3. 使用一次隊長技能。
4. 使用一次補給。
5. 擊敗 Boss。
6. 拾取 COrebound 核心鑰匙或 Tutorial 專用通關物。
7. 觸發通關演出。

### 15.3 完成條件

- Boss 被擊敗。
- 玩家拾取通關物。
- 顯示 Tutorial 通關結算。

### 15.4 下一段預告：通關後解鎖

> 你已完成第一次完整流程。
>
> 下一次，新遊戲會讓你自己選擇是否還需要教學。

---

## 16. 教學第十六段：通關後解鎖

### 16.1 這一段要教什麼

玩家完成 Tutorial 後，要知道自己已經完成教學，下一次可以選擇是否開啟教學。

### 16.2 建議結果

- 標記玩家已有遊玩紀錄。
- New Game 顯示「需要教學嗎？」切換。
- 預設關閉教學。
- 顯示「你已完成 Tutorial，接下來可以嘗試 Easy 或 Normal」。

建議提示：

> 你已完成第一次 Lattice 通關。
>
> 下一次開始新遊戲時，你可以選擇是否再次開啟教學。
