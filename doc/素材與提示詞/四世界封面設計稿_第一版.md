# COrebound LAttence 四世界封面設計稿（第一版）

本文件是給 `主畫面封面 / 商店封面 / 宣傳封面` 使用的四張世界主視覺設計稿。

這一版目標不是直接做 `16 組隊長 x 世界` 完整池，
而是先做出 **四張「世界代表封面」**：

- `幾何世界`
- `有機世界`
- `分形世界`
- `機械世界`

每張圖都代表一個世界本身的情緒、怪物群像、地板語言與神殿氛圍，
並且保留之後再疊加 `四位隊長前景特寫` 的空間。

---

## 共通構圖規則

四張圖全部統一遵守：

1. `橫向構圖`
2. `主視覺偏背景海報感，不是角色立繪單人照`
3. `中央到偏中間區域保留 Logo / Start UI 空間`
4. `怪物群像鋪在後景與左右側，不要把中央塞死`
5. `世界地板一定要出現，讓玩家一眼知道是哪一個世界`
6. `畫面右下或左下放一個該世界的小人圖示`
7. `另外再混入 3 個較小、較淡的隨機圖示當裝飾變體`

## 小人圖示變化規則

目前可直接參考的圖示：

- `幾何`：`assets/images/characters/members/atlases/geometry-icon.png`
- `有機`：`assets/images/characters/members/atlases/organic-icon.png`
- `分形`：`assets/images/characters/members/atlases/fractal-icon.png`
- `機械`：`assets/images/characters/members/atlases/mechanical-icon.png`

建議放法：

- `主圖示`：放一個該世界 icon，尺寸最大，作為該張世界封面的識別章
- `副圖示`：再從四世界 icon 裡隨機抽 3 個，縮小、降透明、散在角落
- `視覺效果`：像戰場 UI 標記、徽章、召喚印記，不要像貼紙亂貼

這樣會有：

- 世界主題很明確
- 每張圖又會有一點隨機感
- 之後真的要做輪播封面時，也比較容易延伸

## 1. 幾何世界封面

### 畫面感受

- 冷靜
- 神聖
- 高秩序
- 像數學文明神殿

### 構圖方向

- 背景以 `晶格地板 + 生命之花 + 超立方投影` 為主
- 左右兩側散開 `圓形 / 三角形 / 正方形 / 正六角形 / 彭羅斯三角 / 生命之花 / 超立方體`
- 中央保留比較乾淨的藍白金光場，讓標題能壓上去
- 右下角放 `geometry-icon`
- 另外三個世界的 icon 用更小、更淡的方式藏在畫面角落

- 主色：`冰藍 / 白 / 金`

## 我對這四張圖的建議節奏

如果你現在是要真的開始做封面，我建議不是四張一起亂衝，而是這樣：

1. 先出 `幾何世界`
2. 確認你喜歡的是「海報感」還是「主畫面留白感」
3. 定調後再複製到另外三個世界

因為四張圖最大的風險不是 prompt 不夠華麗，
而是四張圖最後會變成：

- 各自都很酷
- 但放在一起不像同一款遊戲

所以這份稿刻意把：

- `構圖`
- `Logo 留白`
- `icon 放法`
- `怪物群像鋪法`

全部先鎖住。

---

## 直接可用的第一輪策略

如果你要拿去 AI 生圖，我建議先這樣跑：

1. 每個世界先各生 `2 張`
2. 四世界總共先看 `8 張`
3. 從每個世界挑 `1 張` 最接近方向的
4. 再做第二輪精修

這樣比較不會一次被 20 張圖淹死。
