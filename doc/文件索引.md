## 7. 使用建議

之後如果你還要新增文件，我會建議先問一個問題：

`這份文件比較像是在講什麼？`

然後對應放到：

- 玩法總覽 -> `遊戲總覽`
- 劇情與世界 -> `世界觀`
- 隊長、角色、怪物 -> `角色與敵方`
- 規則、數值、戰鬥 -> `系統機制`
- UI、HUD、按鈕、流程 -> `介面流程`
- 資產、圖片、音樂、提示詞 -> `素材與提示詞`

這樣之後整個文件庫會越來越穩，不會又長回一坨。
