# COrebound LAttence Assets 資料夾總覽

本資料夾用來集中放置 **需要由 AI 生成**，或後續要委託製作的各類素材。

目前依照專案現有規劃，先分成三大類：

1. `images`
2. `video`
3. `audio`

---

## 1. images

放所有靜態圖像資產。

### 目前需要你自己生成的重點

- 隊長與成員立繪
- 怪物與 Boss 立繪 / 圖騰形體
- 世界環境物件
- 礦物與材料圖示
- 地板材質與背景裝飾
- UI 圖示、按鈕、面板視覺
- 地圖縮影與世界預覽圖
- 宣傳圖 / 商店圖 / 封面圖

---

## 2. video

放所有動態影片與動態預覽資產。

### 目前需要你自己生成或製作的重點

- 主畫面或標題動態背景
- UI 動態示意
- 世界展示短片
- 技能展示短片
- 道場 / 新手導覽示範短片
- 宣傳預告 / 商店頁影片

---

## 3. audio

放所有音樂與音效資產。

### 目前需要你自己生成或委託的重點

- 主畫面音樂
- 圖鑑 / 新手導覽 / 訓練道場音樂
- 前中後期戰鬥音樂
- 四大世界主題音樂
- Boss 戰音樂
- 勝利 / 失敗 / 結算音樂
- UI 音效
- HUD 音效
- 技能音效
- 互動物件音效
- 怪物 / Boss / 場景音效

---

## 4. 目前資料夾配置原則

- `images`：依角色、怪物、世界、物件、UI、地圖、宣傳分組
- `video`：依用途分成標題、UI、世界展示、技能展示、預告
- `audio`：分成 `music` 與 `sfx`
