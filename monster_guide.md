# COrebound LAttence 落地版怪獸圖鑑 (Monster Guide)

本文件詳列了 **COrebound LAttence (COLA)** Game Jam 實作落地版中，全部 **29 種敵方圖騰怪物**的數值、戰鬥機制、掉落素材，以及玩家成員的升星合成配方。

> [!IMPORTANT]
> **怪物機制與難度設計規範 (Enemy Mechanics & Difficulty Specification)**
> * **玩家機制照搬與自動施法 (Simulated Auto-Casts & Self-Traits)**：為保持戰鬥邏輯一致，敵方怪物直接照搬玩家的武器子彈、成員多樣性升級中的**自動施法 (Auto-Cast) 與自身被動機制**。
>   * **T0 資源怪** ── 模擬 0★ 初始玩家（無武器，無附魔，無控制引擎，無被動）。
>   * **T1 弱雜兵** ── 模擬 1★ 玩家（使用基礎武器子彈 + **1★ 隊長控制引擎**，無附魔，無被動）。
>   * **T2 精英怪** ── 模擬 2★ 玩家（使用基礎武器 + **2★ 玩家附魔** + **2★ 隊長控制引擎** + **1個玩家成員的2★自動施法/自身被動特徵**）。
>   * **T3 世界 Boss** ── 模擬 4★ 玩家（使用複合武器 + **2個 4★ 玩家附魔** + **4★ 隊長控制引擎終極質變** + **2個玩家成員的4★自動施法/自身被動特徵** + **1個隊長主動位移技能**）。
>   * **T4 最終 Boss** ── 模擬 4★ 滿配完全體玩家（全武器混合 + **多個玩家 4★ 附魔** + **全體 4 個 4★ 控制引擎疊加** + **4世界 4★ 雙被動/自動施法特徵疊加** + **輪流使用隊長主動位移技能**）。
> * **萬花筒對稱生成**：怪物在戰鬥中，在實際渲染時將使用**禪繞畫直線筆法**，以**萬花筒方式**進行繪製，並進行**「六個方位（6-Fold Symmetry，每 60 度對稱映射）」**的鏡像映射效果以快速實作。

## 📊 怪物層級與數值設計規範

在戰鬥中，敵方的子彈與本體碰撞皆適用「碰撞重量機制」。
* **小怪本體重量**：等同於小怪的碰撞耐久度。小怪撞擊玩家防線時，會依據剩餘重量扣減防線重量，並對防線成員造成傷害。
* **無機率確定性**：小怪物的所有技能冷卻與觸發皆以「時間（秒/傷害 Tick）」為確定性週期。
