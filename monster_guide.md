# COrebound LAttence 落地版怪獸圖鑑 (Monster Guide)

本文件詳列了 **COrebound LAttence (COLA)** Game Jam 實作落地版中，全部 **29 種敵方圖騰怪物**的數值與戰鬥機制。本作敵方怪物皆為模仿小隊結構的同心圓禪繞畫圖騰花紋。

> [!IMPORTANT]
> **怪物機制設計規範 (Enemy Mechanics Design Specification)**
> * **玩家機制照搬 (Simulated Player Builds)**：為了簡化程式碼開發難度並保持戰鬥邏輯的一致性，本遊戲的敵方怪物不開發任何獨特的新機制，而是**直接照搬並模擬玩家在不同星級時的武器子彈、成員附魔以及隊長控制引擎**。
>   * **T0 資源怪** ── 模擬 0★ 初始玩家（無武器，無附魔，無控制引擎）。
>   * **T1 弱雜兵** ── 模擬 1★ 玩家（使用基礎武器子彈，無附魔，無控制引擎）。
>   * **T2 精英怪** ── 模擬 2★ 玩家（使用基礎武器 + 1個 **玩家2★附魔** + 1個 **2★隊長控制引擎**）。
>   * **T3 世界 Boss** ── 模擬 3★/4★ 高階玩家（使用複合武器 + **玩家4★附魔** + 1個 **4★隊長控制引擎**）。
>   * **T4 最終 Boss** ── 模擬 4★ 滿配全能玩家（使用複合武器 + **多個玩家4★附魔** + **全體4個4★隊長控制引擎**）。
> * **萬花筒對稱生成**：怪物在戰鬥中，在實際渲染時將使用**禪繞畫直線筆法**，以**萬花筒方式**進行繪製，並進行**「六個方位（6-Fold Symmetry，每 60 度對稱映射）」**的鏡像映射效果以快速實作。

## 📊 怪物層級與數值設計規範

在戰鬥中，敵方的子彈與本體碰撞皆適用「碰撞重量機制」。
* **小怪本體重量**：等同於小怪的碰撞耐久度。小怪撞擊玩家防線時，會依據剩餘重量扣減防線重量，並對防線成員造成傷害。
* **無機率確定性**：小怪物的所有技能冷卻與觸發皆以「時間（秒/傷害 Tick）」為確定性週期。
