# COrebound LAttence — 重製版 (Remake Edition)

**獨立、可完整遊玩、可通關、可重玩**的單人 PVE 重製版。
舊專案(`COrebound-LAttence-main`)僅作為設計文件、素材與資料模組來源,本 repo 為全新施工地。

## 啟動

```bash
npm install
npm run dev        # http://127.0.0.1:5600
```

正式建置:`npm run build` → `dist/`(可用 `npm run preview` 驗收)。

## 驗收

```bash
npx tsc --noEmit    # 型別檢查
npm run test:e2e    # 端到端自動通關測試:bot 直接驅動戰場模擬,
                    # 驗證「敗北路徑」與「完整通關路徑」(約 20~28 分鐘模擬時間,實際執行數秒)
```

## 完整遊玩路線(通關方式)

1. **主畫面**(四世界舞台影片輪播)→ 開始遊玩 → 四選一隊長(Conductor/Operator/Launcher/Architect,主動技與控制效果各異)。
2. **發育期**:WASD 探索四象限世界(幾何/有機/分形/機械),擊殺 T0/T1 怪收集原石與生物材料。
3. **建隊**:在各世界的**成員雕像**供奉材料解鎖成員(每世界 5 名、跨 5 家族;同家族湊 2 人自動啟用該家族武器)。
4. **成長**:在**工坊**熔多餘材料 → 家族碎片,用 5-3-1 配方為成員升星;小隊總星 5/10/15 時隊長自動進化,2★ 起全彈附帶控制效果。**商店**買藥水(1~4 鍵使用)。
5. **推進**:每世界擊殺雜兵 ×10、核心區精英 ×2 後,到**守護者祭壇**召喚 T3 世界守護者;擊敗得**晶核印記**,該世界狂暴、其他世界連動強化。
6. **決戰**:集齊 4 枚印記 → 回**中央廣場 COLA 裝配儀** → 召喚最終 Boss **COLA** → 擊敗即勝利。
7. **壓力**:第 10 分鐘起晶格侵蝕向核心收縮(圈外 5%/秒);死亡扣 30% 原石、廣場復活,**第 3 次死亡即敗北**。
8. **結算**:勝敗皆有完整戰報 → 再來一場 / 回主畫面。

操作:`WASD` 移動 · `Space` 隊長主動技 · `E` 互動 · `Tab` 小隊面板 · `1~4` 藥水 · `Esc` 關閉面板。

## 與舊專案的關係

- 吸收的設計決策、抽取的程式模組、重用的素材:見 [AUDIT.md](AUDIT.md)。
- `src/legacy/` 內為舊專案原樣搬運的資料庫與純函式(成員 20 名、怪物 29 種、武器/附魔/控制/素材數值、隊長主動技效果)。
- `public/assets/` 內為舊專案素材:4 隊長立繪、60 張成員頭像、29 種敵方圖騰、4 張世界地板、設施與環境物件、5 首音樂、4 支世界影片。

## 重製版新增/調整(相對舊文件)

- 單人通關制(勝=殺 COLA,敗=三次全滅);不做 PVP 網路對戰。
- 守護者配額 10/2(文件 15/3)、保護期 5 分(文件 10 分)、侵蝕 10 分起且收縮下限保住世界核心區。
- 小隊生命合池;熔爐+合成台合併為「工坊」;脫戰 6 秒後每秒回血 2.5%;開局與復活短暫無敵。
- Boss 世界韌性:守護者 HP ×2.2、COLA ×1.35、近身接觸極痛(逼走位)、小隊碰撞對 Boss 折減(逼武器輸出)、武器自動集火射程內 Boss。
