# COrebound LAttence 隊長立繪共通 Prompt 與四星提示詞

本文件是提供給之後反覆生成隊長立繪時使用的專用提示詞工作頁。

用途很單純：

- 放 `隊長立繪的共通 Prompt`
- 放 `四位隊長各自的角色 Prompt`
- 放 `四星隊長版本` 的強化 Prompt

這份文件適合在每次要做新版本立繪時直接打開，
不用每次回頭從多份文件重新拼湊。

## 1. 隊長立繪共通 Prompt Base

這一段是所有隊長立繪都可以共用的基底。

建議用法：

- 先貼上這段
- 再加上各隊長專屬 Prompt
- 最後再補當次你想要的表情、姿勢、鏡頭或版本差異

## 3. 隊長立繪共通補充規則

每次補詞時，建議記得維持這幾件事：

1. `輪廓先清楚`
2. `不要讓配件比人還搶`
3. `同時保留幻想感與系統感`
4. `隊長要像角色，不要像怪物`
5. `四星版本要偏完整、偏終極、偏接近 COLA`

## 5. 四位隊長四星版設計原則

因為隊長最高是 `4 星`，
所以四星版不應只是「衣服變華麗」。

四星版應該表現出：

- 更接近完整人格
- 更接近世界核心
- 更接近 COLA 的收斂狀態

所以四星立繪建議共同補上這些概念：

- `awakened final form`
- `four-star captain`
- `core resonance`
- `evolved sacred lattice`
- `greater ornamental complexity`
- `stronger halo / core motif`
- `more complete silhouette`

可作為四星共通補充尾詞：

```text
four-star awakened captain form, evolved sacred lattice motifs, stronger concentric halo, higher-order ornament detail, core resonance visible, closer to a complete world-being, ultimate but still elegant
```

## 7. 每次出圖時可加的可變欄位

- `front view`
- `three-quarter view`
- `battle-ready pose`
- `calm standing pose`
- `story illustration pose`

- `side profile`
- `glowing eyes`

## 8. 建議使用方式

最實用的流程是：

1. 先選隊長
2. 先選一般版或四星版
3. 貼上共通 Base
4. 貼上該隊長 Prompt
5. 最後補本次鏡頭、表情、姿勢、版本差異

這樣你之後每次要做新的隊長立繪，不會越做越亂，風格也比較容易穩住。
